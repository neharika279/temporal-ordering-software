Temporal Ordering of Sampled Data Readings for Biological Data

This is a web-based application that makes available different techniques to order data samples temporally. Any of these methods can be used for data that has been sampled at various points in time, where the exact time point of sampling in unknown. this can be used in the following way:

  • Upload file containing data samples, without sampling times, in a valid format
  • Select the temporal ordering approach to be used on the input data
  • Obtain an ordering of the data samples. The ordering is given in the form of numbers for the data samples that is based on their order in the input file. The results are displayed graphically as well

These methods can be used for biological data such as gene expressions, mRNA profiling experiments, cell lineage tracing. Although the algorithms implemented here were primarily constructed for biological data, it can find applications in any kind data that follows a chronological order. For instance, archeological samples. This is because, the main principle behind all these methods, is to find the distance or level of dissimilarity between data points, and use that to predict the closeness of the sampled data points with respect to time. This is assuming the data is for a process that progresses or changes as a function of time.

File Format

The input file provided must either be a tab seperated file, or a comma seperated file. Each new line of the file would denote an observation/sampling done in an unknown time point. Each tab or comma seperated value in every line will indicate the values of the biological variables recorded for that instance of sampling
