Reconstructing the temporal ordering of biological samples

This algorithm reconstructs the order of the input microarray gene data This is done by construction of a minimum spanning tree that is based on an nxn euclidean distance matrix where n is the number of samples. The matrix gives the Euclidean distances between gene expressions for each pair of samples. Upload a data file containing gene expressions of n number of samples to see the temporal ordering.
